# E-Commerce Sales Analytics Dashboard (Power BI)

## Overview
This project analyzes an e-commerce sales dataset to understand sales performance,
profitability, and customer trends using Power BI.

## Tools Used
- Power BI
- DAX
- Excel
- SQL (for data understanding)

## Key KPIs
- Total Sales
- Total Profit
- Total Quantity
- Profit Margin
- Year-to-Date (YTD) metrics

## Dashboard Insights
- Technology category generated the highest profit
- West and Central regions contributed the most to total sales
- Standard shipping mode had the highest order volume
- Seasonal sales trends with peak performance in Q4
